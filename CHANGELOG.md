# Changelog

All notable changes to OpenClaw Ops Playbook will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-28

### Added

- **Agent Foundation** — AGENTS, SOUL, IDENTITY, BOOTSTRAP, and USER templates
- **Domain Routing** — ROLES template with 7-domain routing architecture
- **Memory System** — MEMORY template with daily notes + long-term curated memory pattern
- **Family Domain** — Overview, per-child, partner, extended family, pets, and schools templates
- **Household Domain** — Vendors, maintenance, vehicles, subscriptions, and insurance templates
- **Work Domain** — Overview, main job, side gig, and knowledge base templates
- **Health Domain** — Family medical and fitness templates
- **Finances Domain** — Budget, investments, and tax prep templates
- **Travel Domain** — Profiles and upcoming trip templates
- **Hobbies Domain** — Per-hobby template
- **People & Automation** — CRM, groceries, birthdays, emergency contacts, heartbeat, and tools templates
- **Scripts** — Driving ETA via Google Directions API
- **Full Playbook** — Complete walkthrough with architecture, patterns, and examples
- **Marketing Site** — Single-page landing site with Jekyll support for GitHub Pages
